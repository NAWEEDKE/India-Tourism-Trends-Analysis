**India Tourism Trends Analysis (2010–2019)**

📌 *Project Overview*

This project provides a comprehensive analysis of tourism trends across various Indian monuments over a decade (2010–2019). By leveraging data manipulation, visualization, and machine learning, the project identifies seasonal patterns, growth trends, and the primary factors influencing visitor footfall.

The core of the project is a predictive model that estimates future visitor numbers with high accuracy, assisting in better crowd management and resource allocation for heritage sites.

🛠️ *Tech Stack*

Language: Python

Libraries: Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn

Visualization: Tableau (Desktop/Public)

Machine Learning: Random Forest Regressor, K-Means Clustering

📊 *Key Features*

Exploratory Data Analysis (EDA): Deep dive into 10 years of monthly tourism data to find the most visited circles (Agra, Delhi, etc.) and monuments (Taj Mahal, Agra Fort).

Seasonality Analysis: Identification of peak months for domestic vs. foreign tourists.

Predictive Modeling: A Random Forest Regression model used to forecast visitor counts based on Monument ID, Circle, and Time.

Interactive Dashboards: Tableau workbooks providing a visual representation of geographical tourism distribution.

📈 *Results & Insights*

Model Accuracy: The Random Forest model achieved an R² Score of 0.98, indicating 98% accuracy in predicting footfall.

Primary Drivers: "Monument ID" was identified as the most significant factor in predicting visitor numbers, followed by "Circle ID" and "Month."

Domestic vs. Foreign: Domestic tourism shows steady year-round growth, while foreign tourism is highly sensitive to seasonality (peaking in winter months).
